Colors

Text color

The template defines 9 text colors: class="white", class="silver", class="grey", class="anthracite", class="black", class="red", class="orange", class="green" and class="blue"

Icons color

By default, icons inherit from text color, but if you want to set a different color there is a similar set of color classes:

class="icon-white", class="icon-silver", class="icon-grey", class="icon-anthracite", class="icon-black", class="icon-red", class="icon-orange", class="icon-green" and class="icon-blue"

Background colors

The template is also bundled with a set of 9 background colors, each one available as a simple tone, a fine gradient and a glossy gradient. Every color class includes a background color, a border color and a hover effect for links. Of course, these colors may be used on any element!

Simple colors

class="white-bg" class="silver-bg" class="grey-bg" class="anthracite-bg" class="black-bg" class="red-bg" class="orange-bg" class="green-bg" class="blue-bg"

Gradients

class="white-gradient" class="silver-gradient" class="grey-gradient" class="anthracite-gradient" class="black-gradient" class="red-gradient" class="orange-gradient" class="green-gradient" class="blue-gradient"

Glossy gradients

class="white-gradient glossy" class="silver-gradient glossy" class="grey-gradient glossy" class="anthracite-gradient glossy" class="black-gradient glossy" class="red-gradient glossy" class="orange-gradient glossy" class="green-gradient glossy" class="blue-gradient glossy"

Textured backgrounds

Several textured background styles are available, for use on a block or event the whole document:

class="carbon"

class="linen"

class="scratch-metal"
